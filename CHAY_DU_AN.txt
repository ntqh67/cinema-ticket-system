HUONG DAN CHAY DU AN CINEMA TICKET SYSTEM
========================================

Thu muc du an:
/home/ntqh/Desktop/Project/cinema-ticket-system


1. KHOI DONG POSTGRESQL VA REDIS
--------------------------------

Truoc tien, kiem tra dich vu da chay san chua:

  pg_isready -h localhost -p 5432
  redis-cli -h localhost -p 6379 ping

Neu ket qua la:

  localhost:5432 - accepting connections
  PONG

thi PostgreSQL va Redis da hoat dong. BO QUA cac lenh Docker ben duoi va
chuyen thang den buoc 2.

Neu may co Docker Compose:

  docker compose up -d

Neu may dung ban Docker Compose cu:

  docker-compose up -d

Neu ca hai lenh tren deu bao "unknown shorthand flag" hoac "command not
found" nhung pg_isready van accepting connections va Redis van PONG, ban
khong can cai Docker Compose de chay du an.

Kiem tra lai dich vu sau khi khoi dong Docker:

  pg_isready -h localhost -p 5432
  redis-cli -h localhost -p 6379 ping


2. TAO FILE MOI TRUONG
----------------------

Neu chua co file .env, chay tai thu muc goc:

  cp .env.example .env

Windows CMD:

  copy .env.example .env

Mac dinh database local:

  DATABASE_URL="postgresql://postgres:postgres@localhost:5432/cinema_ticket_system?schema=public"


3. CAI DEPENDENCY
-----------------

Tai thu muc goc:

  npm install

Sau do cai backend:

  cd backend
  npm install
  cd ..


4. CHUAN BI DATABASE
--------------------

Tai thu muc goc:

  npx prisma validate
  npx prisma generate
  npx prisma migrate deploy

Chi seed khi database dang trong hoac muon tao lai du lieu demo:

  npx prisma db seed

Khong chay seed neu da import file:

  database/cinema_ticket_system_dump.sql


5. CHAY BACKEND
---------------

Mo Terminal 1:

  cd backend
  npm run start:dev

Backend se chay tai:

  API:     http://localhost:3000/api
  Health:  http://localhost:3000/api/health
  Swagger: http://localhost:3000/api/docs


6. CHAY FRONTEND
----------------

Mo Terminal 2 tai thu muc goc du an.

Lua chon 1 - Vite:

  npx vite frontend --host 0.0.0.0 --port 5173

Lua chon 2 - Python (neu may da co Python 3):

  python3 -m http.server 5173 --directory frontend

Windows co the dung:

  py -m http.server 5173 --directory frontend

Mo trinh duyet:

  http://localhost:5173

8. DUNG DU AN
-------------

Tai terminal backend va frontend, nhan:

  Ctrl + C

Dung PostgreSQL va Redis Docker:

  docker compose down

Khong dung "docker compose down -v" neu muon giu lai du lieu database.


9. XU LY LOI THUONG GAP
-----------------------

Backend khong ket noi duoc database:

  - Kiem tra PostgreSQL dang chay o cong 5432.
  - Kiem tra DATABASE_URL trong file .env.
  - Kiem tra user/password PostgreSQL la postgres/postgres.

Frontend khong goi duoc API:

  - Dam bao backend dang chay o cong 3000.
  - Mo frontend bang http://localhost:5173, khong mo truc tiep file index.html.
  - Nhan Ctrl + F5 de xoa cache JavaScript cu.

Redis khong ket noi duoc:

  - Kiem tra Redis dang chay o cong 6379.
  - Chay: redis-cli ping
  - Ket qua dung phai la: PONG

Cong 3000 hoac 5173 da duoc su dung:

  Linux/macOS:
    lsof -i :3000
    lsof -i :5173

  Windows:
    netstat -ano | findstr :3000
    netstat -ano | findstr :5173
