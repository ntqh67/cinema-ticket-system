HUONG DAN CHAY DU AN CRTicket
=============================

Nhanh hien tai: merge-main2-with-445-db
Database dump dung de chia se: database/cinema_ticket_system_dump.sql
Frontend: http://localhost:5173
Backend API: http://localhost:3000/api
Swagger: http://localhost:3000/api/docs

Luu y quan trong:
- File .env KHONG duoc push len Git vi co secret SePay/TMDB.
- Nguoi pull ve can tao .env tu .env.example roi dien gia tri that neu muon test SePay/TMDB.
- Neu muon database giong het may Huy tai thoi diem push, phai import file dump, KHONG chay prisma db seed sau do.

3. TAO FILE .env
================

Neu chua co .env:

copy .env.example .env

Gia tri local toi thieu:

DATABASE_URL="postgresql://postgres:postgres@localhost:5432/cinema_ticket_system?schema=public"
FRONTEND_URL="http://localhost:5173"
BOOKING_HOLD_MINUTES="10"

Neu muon test SePay that, dien them:

SEPAY_ENABLED="true"
SEPAY_BANK_ACCOUNT="<SO_TAI_KHOAN_NGAN_HANG>"
SEPAY_BANK_CODE="<MA_NGAN_HANG_VI_DU_MBBank>"

Neu khong test SePay:

SEPAY_ENABLED="false"


4. BAT DOCKER POSTGRES VA REDIS
===============================

docker compose up -d
docker compose ps

Can thay 2 container dang Up:
- postgres
- redis


5. XOA DATABASE CU VA IMPORT DUMP CUA NHANH
===========================================

Canh bao: cac lenh duoi day xoa toan bo database local hien tai.

docker compose exec -T postgres psql -U postgres -d cinema_ticket_system -c "DROP SCHEMA public CASCADE; CREATE SCHEMA public;"
Get-Content database\cinema_ticket_system_dump.sql | docker compose exec -T postgres psql -U postgres -d cinema_ticket_system -v ON_ERROR_STOP=1
docker compose exec redis redis-cli FLUSHDB
npx.cmd prisma generate
npx.cmd prisma migrate status

Sau khi import dump:
- KHONG chay npx.cmd prisma db seed
- KHONG chay npx.cmd prisma migrate reset

Neu chay seed/reset thi database se khong con giong file dump nua.

8. CHAY NGROK CHO SEPAY WEBHOOK
===============================

Mo terminal PowerShell thu ba:

Lay Forwarding HTTPS, vi du:

https://xxxxx.ngrok-free.dev

Cap nhat .env:

SEPAY_WEBHOOK_URL="https://xxxxx.ngrok-free.dev/api/bookings/sepay-webhook"

Sau khi sua .env, phai restart backend.

Kiem tra ngrok:

https://xxxxx.ngrok-free.dev/api/bookings/payment-methods

Luu y:
- Mo thang https://xxxxx.ngrok-free.dev/ bi Cannot GET / la binh thuong.
- Ngrok dang tro toi backend API, khong phai frontend.

13. XUAT LAI DATABASE DUMP SAU KHI SUA DU LIEU
==============================================

Chi dung khi muon dong goi database local hien tai de day len Git:

cmd /c "docker compose exec -T postgres pg_dump -U postgres -d cinema_ticket_system --clean --if-exists --no-owner --no-privileges --encoding=UTF8 > database\cinema_ticket_system_dump.sql"

Sau do commit file:

git add database/cinema_ticket_system_dump.sql run.txt
git commit -m "db: update packaged database dump"


14. LOI THUONG GAP
==================

SePay bao "bank account is not configured":
- Kiem tra .env co SEPAY_ENABLED=true va SEPAY_BANK_ACCOUNT khong rong.
- Restart backend sau khi sua .env.

Ngrok hien "Cannot GET /":
- Binh thuong, vi ngrok tro toi backend.
- Kiem tra bang /api/bookings/payment-methods.

Anh rap khong dung:
- Bam Ctrl + F5.
- Xoa localStorage key cineticket_cinemas.

Port 3000 dang bi chiem:

netstat -ano | findstr :3000
taskkill /PID <PID> /F

Port 5173 dang bi chiem:

netstat -ano | findstr :5173
taskkill /PID <PID> /F
